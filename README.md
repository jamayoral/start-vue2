Starting with VueJS2

- npm install -g http-server

- Init examples: 
    - cd start-vue2
    - http-server -c-1
    - http://localhost:8080/codigo
    
- Run Vue-cli example: 
    - cd webapp
    - npm install
    - npm run dev
    - http://localhost:8080/home
